# **App Name**: Eventi Sahel

## Core Features:

- User Account Management: Users can create and manage their profiles, including personal information and purchase history.
- Event Browsing: Users can browse and search for events in the Sahel region, filtering by category, date, and location.
- Ticket Purchasing: Users can purchase tickets for events online, with secure payment processing.
- Admin Dashboard (Event & Pricing): Admins can manage events, set prices, and view sales data.
- Gate Dashboard (QR Code Verification): Personnel at the gate can scan QR codes on tickets to verify entry.
- Super Admin Dashboard: Super admins can set up events, manage commissions, and view financial reports.

## Style Guidelines:

- Primary color: red , evocative of Sahel sunsets, for an energetic feel.
- Background color: Light grey (#E0E0E0), to provide contrast and allow event visuals to pop.
- Accent color: Gold (#FFD700) and to symbolize Sahel's cultural richness and highlight important elements.
- Body and headline font: 'Inter', a versatile sans-serif providing clarity for event information and user interface elements.
- Use clean and modern icons for navigation and event categories.
- A clear and intuitive layout is used throughout the app to simplify event browsing and ticket purchasing.
- Subtle animations, such as smooth transitions and loading indicators, enhance the user experience without being distracting.
